# QR Code Generator

A simple, fast, and fully client-side QR Code Generator built with HTML, CSS, and JavaScript.  
No backend, no tracking, no data storage — everything runs directly in your browser.

---

## 🚀 Features

- Generate QR codes instantly from text or URLs
- Fully responsive and lightweight design
- No installation required
- Works on GitHub Pages
- Privacy-friendly (nothing is stored or sent anywhere)
